# Problem 5: Print the First N Multiples of a Number

### Problem Description

Write a program that takes a number X and a count N as input. Print the first N multiples of X. (Multiplication table)
### Input

- Enter a number: 5
- Enter the count: 10

### Output

5 x 0 = 0
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
